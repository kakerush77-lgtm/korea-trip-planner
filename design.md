- 見出し: System Bold (iOS SF Pro / Android Roboto)

- 数字: System Monospaced (ポイント表示等)

## 画面リスト

### 認証フロー
1. **WelcomeScreen** - アプリ紹介・ログインボタン
2. **LoginScreen** - Manus OAuth認証（Google/Apple/LINE/メール）

### メインタブ (4タブ)
3. **HomeScreen** - ダッシュボード（次の旅行、ポイント、おすすめ）
4. **TripsScreen** - しおり一覧（検索・フィルター付き）
5. **MyPageScreen** - プロフィール・設定・実績

### サブ画面
6. **TripDetailScreen** - 旅行詳細（予定/タイムライン/リスト/割り勘/メンバー）
7. **CreateTripFlow** - 旅行作成ウィザード（3ステップ）
8. **AddSpotScreen** - スポット追加
9. **PointsScreen** - ポイント・ミッション・ショップ
10. **NotificationsScreen** - 通知一覧

## 主要コンテンツと機能

### HomeScreen
- ウェルカムカード（ユーザー名・ポイント表示）
- 次の旅行カウントダウンカード
- 出発前チェックリスト
- 予算トラッカー
- 通貨換算クイックルック
- ポイントCTA（貯める/使う）
- おすすめ記事フィード

### TripsScreen
- 検索バー
- しおりカード一覧（カバー色・メンバーアバター・スポット数）
- 空状態表示

### TripDetailScreen
- 5つのサブタブ: 予定 / タイムライン / リスト / 割り勘 / メンバー
- 日別スポット表示（カテゴリフィルター付き）
- スポットカード（時間・カテゴリ・メモ・移動情報・予約情報）
- リスト管理（買い物/リンク/持ち物）
- 割り勘計算・精算サマリー

### MyPageScreen
- プロフィールヒーロー（アバター・ランク・ポイント）
- 統計（しおり数・スポット数・訪問国数）
- 実績バッジ
- 設定メニュー

## キーユーザーフロー

### 認証フロー
1. アプリ起動 → WelcomeScreen表示
2. 「ログイン」タップ → OAuth認証画面
3. 認証完了 → HomeScreenへ遷移

### 旅行作成フロー
1. HomeまたはTrips画面で「+」タップ
2. Step 1: タイトル・国選択
3. Step 2: 日程選択
4. Step 3: カバー色・地図プロバイダー選択
5. 完了 → TripDetail画面へ遷移

### スポット追加フロー
1. TripDetail「予定」タブで「+」タップ
2. カテゴリ選択（食事/観光/ショッピング等）
3. スポット名・時間・メモ入力
4. 移動情報・予約情報（オプション）
5. 保存 → タイムラインに反映

## レイアウト原則

- 全画面: ScreenContainer使用（SafeArea対応）
- カード: 角丸16-22px、軽い影
- ボタン: 角丸10-14px、タップフィードバック（scale 0.97）
- テキスト: 見出し16-18px、本文10-12px、サブ8-9px
- 間隔: セクション間16px、要素間8px
- タブバー: 4タブ（ホーム/しおり/マイページ + 中央に作成ボタン）
